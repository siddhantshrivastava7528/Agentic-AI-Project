# ⚛️ Physics Study Buddy Agent

An AI-powered Physics assistant built using **LangGraph, RAG (ChromaDB), and Groq LLM**.

---

## 🚀 Features
- 🧠 Agentic AI (LangGraph)
- 📖 RAG with ChromaDB
- 🔧 Tool support (Web search)
- 💬 Conversation memory
- ⚡ Streamlit chat UI

---

## 🏗️ Architecture
User → Memory → Router → (Retrieve / Tool / Skip) → Answer → Save

---

## 🛠 Tech Stack
- Python
- Streamlit
- Groq (LLaMA)
- SentenceTransformers
- ChromaDB
- LangGraph
